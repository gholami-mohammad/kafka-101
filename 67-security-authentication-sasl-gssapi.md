# احراز هویت با استفاده از SASL

SASL = Simple Authentication Security Layer

احراز هویت sasl در کافکا به چند صورت پشتیبانی میشود:

- روش PLAIN: کلاینت با استفاده از یک نام کاربری و رمز عبور احراز هویت میکند. (ضعیف - تنظیمات آسان)
- روش GSSAPI: مانند استفاده از Kerberos یا Microsoft Active Directory (قوی - تنظیمات دشوار)
- روش SCRAM: نام کاربری و رمز عبور (قوی - تنظیمات نسبتا دشوار)
- روش OAUTHBEARER: (قوی - تنظیمات دشوار)

در حال حاضر روش GSSAPI محبوب ترین روش است.

در ادامه هر ۴ روش به دقت پیاده سازی خواهد شد.
